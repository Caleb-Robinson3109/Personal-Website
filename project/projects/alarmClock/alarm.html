GUI Application - An Alarm Clock

Caleb Robinson

April 2022

Introduction

In my C++ II class at St. Louis Community College the final projects was to make a gui applacation. For this application I chose an alarm clock. The alarm clock has three base features. 1 - tell time, 2 - set an Alarm, 3 - snooze the alarm. The final application has one more feature: the ability to check what time the alarm is set at.

Technologies Used

  • C++
  • wxWidgets
