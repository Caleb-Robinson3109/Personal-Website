Pickaxe Protection - PickHacks Hackathon

Caleb Robinson

April 2024

Introduction

The first Hackathon that I attended was PickHacks. In this Hackathon my group and I created Pickaxe Protection! This is a web app that helps students find lost items. If a pair of AirPods where lost, then someone could report it on the app. The person who lost the AirPods could see where they where found, and claim them.

Technologies Used

  • React
  • Typescript
  • Java & Springboot
  • mySQL

Results

New Technologies were learned, this was the first time I had used React, and Pickaxe Protection was born. There was a lot of trial and error, but in the end we finished, and it dosn't work half bad!
